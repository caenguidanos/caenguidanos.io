## Install

```bash
# assign node version to terminal session
nvm use 16.13.0
# install pnpm
npm i -g pnpm
# install project dependencies
pnpm i
```

## Run

```bash
pnpm dev
```

## Testing

### Storybook

Run **Storybook** server:

```bash
pnpm storybook
```

### Unit

Run **Jest** tests:

```bash
pnpm test
```

### E2E

Run **Playwright** tests:

```bash
pnpm e2e
```
