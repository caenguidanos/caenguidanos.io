## Install

```bash
nvm use
npm i -g pnpm
pnpm i
```

## Run

```bash
pnpm dev
```

## Testing

### Unit

Run **Jest** tests:

```bash
pnpm test
```

### E2E

Start desired server:

```bash
pnpm dev
```

Run **Playwright** tests:

```bash
pnpm e2e
```
